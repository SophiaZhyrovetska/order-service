# Order service
Simple service for making purchases with NestJS and PostgreSQL.
## How to run

Create *.env* file in the root directory and provide the following

```sh
DB_HOST=''
DB_PORT=
DB_NAME=''
DB_USERNAME=''
DB_PASSWORD=''
REPORT_SERVICE_URL=''
ANALYTICS_SERVICE_URL=''
```

Install the dependencies and start the server

```sh
$ npm install
$ npm run start
```
